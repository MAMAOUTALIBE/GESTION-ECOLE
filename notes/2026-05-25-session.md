## 🚀 Comment relancer l'app demain

### 1) Vérifier que Postgres + Redis tournent

```bash
lsof -iTCP:5433 -sTCP:LISTEN  # Postgres
lsof -iTCP:6379 -sTCP:LISTEN  # Redis
```

S'ils sont arrêtés, démarre-les via :
- **Postgres** : Docker Desktop (si tu as une image postgis) OU `brew services start postgresql@16`
- **Redis** : `brew services start redis` OU `redis-server`

### 2) Lancer le backend (terminal 1)

```bash
cd /Users/bahmamadou/Desktop/GESTION-EE/Backend
.venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

Attends `Application startup complete` puis vérifie :
```bash
curl http://localhost:8000/ready
# devrait renvoyer : {"database":"ok","redis":"ok"}
```

### 3) Lancer le frontend (terminal 2)

```bash
cd /Users/bahmamadou/Desktop/GESTION-EE/Final
npm run start:node20
```

Attends `Local: http://localhost:4200/` puis ouvre cette URL.
